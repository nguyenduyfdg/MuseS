Description

The project show the design and manufacture of a small wearable device that can be attached on a motorcycle or bicycle to instruct the driver's direction.

The device will be loaded which was processed data before starting the journey and operate completely independent without connecting to any other mobile device. Based on the coordinated data which was collected from the GPS module and the data from mobile device, SmártGo will display the direction for the user.
